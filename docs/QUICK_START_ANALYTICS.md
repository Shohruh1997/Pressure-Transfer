## 📊 Подключение Google Analytics 4 (на следующей неделе)

### Шаг 1: Создать аккаунт Google Analytics

1. Откройте [Google Analytics](https://analytics.google.com/)
2. Нажмите **"Начать"** или **"Создать аккаунт"**
3. Заполните информацию об аккаунте:
   - Имя аккаунта: `Flex Convert`
   - Сайт: `https://flex-convert.ecomdev.uz`

### Шаг 2: Создать свойство (Property)

1. Нажмите **"Создать свойство"**
2. Введите имя свойства: `Flex Convert`
3. Выберите часовой пояс и валюту
4. Нажмите **"Создать"**

### Шаг 3: Получить ID измерения (Measurement ID)

1. В левом меню перейдите в **"Администратор"** → **"Источники данных"** → **"Потоки данных"**
2. Выберите свой вебсайт
3. Скопируйте **ID измерения** (выглядит как `G-XXXXXXXXXX`)

### Шаг 4: Добавить код в сайт

1. Откройте файл `index.html`
2. Найдите строку с `GA_MEASUREMENT_ID`:
   ```html
   <script async src="https://www.googletagmanager.com/gtag/js?id=GA_MEASUREMENT_ID"></script>
   ```

3. Замените `GA_MEASUREMENT_ID` на ваш ID (например, `G-XXXXXXXXXX`):
   ```html
   <script async src="https://www.googletagmanager.com/gtag/js?id=G-XXXXXXXXXX"></script>
   ```

4. Также замените в конфиге:
   ```javascript
   gtag('config', 'G-XXXXXXXXXX', { anonymize_ip: true });
   ```

### Шаг 5: Проверить установку

1. Сохраните файл и обновите сайт в браузере
2. Откройте Google Analytics → **"Отчёты в реальном времени"**
3. Перезагрузите сайт несколько раз
4. Должны появиться данные о вашем посещении

## ❓ Часто Задаваемые Вопросы

**Q: Почему нужна аналитика?**  
A: Без данных вы слепы. Analytics показывает, откуда идут пользователи, какие страницы популярны и как их удержать.

**Q: Когда смогу заработать?**  
A: AdSense платит, когда трафик > 10K просмотров/месяц. Реалистично это 2–4 месяца усердной работы.

**Q: Какой минимум нужно инвестировать?**  
A: $0–$200/месяц. Основное — ваше время (10–20 часов в неделю на контент).

**Q: Почему не все статьи сразу?**  
A: Минимальный подход. Лучше 3 хороших статьи, чем 10 плохих. Отслеживайте, что работает, потом масштабируйте.
