# Конвертер давления (Pressure Converter)

Современное статичное веб-приложение для быстрой и точной конвертации различных единиц измерения давления. Проект переписан с устаревшего десктопного приложения на C++ (Qt) в удобный веб-формат.

## Поддерживаемые единицы измерения
- **Па** (Паскаль)
- **кПа** (Килопаскаль)
- **МПа** (Мегапаскаль)
- **Бар**
- **кгс/см²** (Техническая атмосфера)
- **атм** (Физическая атмосфера)
- **мм рт. ст.** (Миллиметры ртутного столба)

## Технологии
- **HTML5**
- **JavaScript (ES6+)**
- **Tailwind CSS** (через CDN для стилизации)
- Полностью статичное приложение без зависимостей (не требует Node.js или сборщиков).

## Особенности
- Мгновенная конвертация "на лету" при вводе значений.
- Единая математическая модель через базовую единицу (Паскаль), что исключает ошибки перекрестной конвертации.
- Адаптивный и современный интерфейс.

## Использование (Локально)
Просто скачайте репозиторий и откройте файл `index.html` в любом современном браузере.
