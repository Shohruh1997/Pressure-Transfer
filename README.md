# Flex Convert

Корневой сайт проекта — единый статический SPA-shell, в котором категории подключаются через конфигурацию и адаптеры.

## Что уже работает

- Категория Давление на общем движке конвертации через базовую единицу Па.
- Категория Температура на том же UI через отдельный формульный адаптер.
- Верхняя навигация между категориями без перезагрузки страницы.
- Переключение темы.
- Универсальные карточки результатов с копированием.
- Валютная вкладка с локальными курсами, настройкой активных валют и отдельным калькулятором.

## Текущий стек

- HTML5
- JavaScript ES modules
- Tailwind CSS через CDN
- Статический запуск без сборки и без собственного бэкенда

## Структура проекта

## Локальный запуск

Запускайте проект через любой локальный HTTP-сервер и открывайте корневой `index.html` в браузере.

## Ближайшие шаги

1. Подключить живые курсы валют отдельным сервисным слоем.
2. Вынести CSS из `index.html` в отдельный файл.
3. Добавить точечные тесты на конвертацию и валютный калькулятор.
